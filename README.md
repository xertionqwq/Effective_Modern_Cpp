# Effective-Modern-Cpp-
## 一.简介
主要针对要学习现代cpp的人, 通过**effective modern c++**书籍以及拓展的基本知识和相关知识.
### 使用要求
1. 掌握基本c++语法, 理解并使用基本的**类与对象**,基本的**stl**使用.
2. 会用最简单的cmake进行项目构建
3. 在本地电脑下载cmake, 以便使用cmake

## 二.如何使用
由于effective modern c++实在是写的很乱, 对于初学者来说相当地眼花缭乱, 书中忽略了大量的**前置**知识, 导致学习中需要不断补充相关基础. 再就是条款其实没分先后, 导致不知道从何入手, 看哪都不懂, 所以本仓库诞生.
