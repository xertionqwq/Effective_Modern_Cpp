# Effective-Modern-Cpp-

## 一.简介
主要针对要学习现代cpp的人, 通过**effective modern c++**书籍以及拓展的基本知识和相关知识.

### 使用要求
1. 掌握基本c++语法, 理解并使用基本的**类与对象**,基本的**stl**使用.
2. 会用最简单的**cmake**进行项目构建
3. 在本地电脑下载cmake, 以便使用cmake

## 二.如何使用
由于effective modern c++实在是写的很乱, 对于初学者来说相当地眼花缭乱, 书中忽略了大量的**前置**知识, 导致学习中需要不断补充相关*基础*. 再就是*条款*其实没分先后, 导致不知道从何入手, 看哪都不懂, 所以本仓库诞生.

### 叠甲
1. 本仓库大多数参考[b站视频:【Effective Modern C++】啃书第一章：类型推导](https://www.bilibili.com/video/BV1Gg4y1p71w/?spm_id_from=333.1387.collection.video_card.click&vd_source=58df4c541602fc2c9ac62bb34a0652f9)
可结合视频进行使用.
2. 第一次构建项目, 先删除*build*文件夹, 再点击vscode下方状态栏:Gear:*生成*(或者*build*), 等待项目第一次生成.
3. 如果发现有报错, 请手动定位到相关文件, 将相关代码注释掉~~一般不是很多~~(使用**ctrl+/**可一键注释当前内容)
4. 如果要运行项目, 请点击vscode的*CMake*一栏, 再*调试*下方可看见当前正在运行的项目, 点击右边的:fountain_pen:*设置启动/调试目标*, 并在接下来的框框内选择对应项目~~选什么总知道吧~~, 接着点击下方状态栏的正三角形*启动所选目标*
